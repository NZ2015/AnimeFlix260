# AnimeFlix26 - Plateforme de Streaming d'Anime

Une page d'accueil moderne et responsive pour une plateforme de streaming d'animes, créée avec HTML5, CSS3 et JavaScript vanilla.

## 🎨 Caractéristiques

### Design Moderne
- Interface sombre élégante avec gradients vibrants
- Palette de couleurs: Rose/Magenta, Violet, Bleu
- Animations fluides et transitions douces
- Design responsive sur tous les appareils

### Composants Principaux

#### Header
- Logo animé avec icône de lecture
- Barre de navigation avec liens actifs
- Boutons d'authentification (Connexion/Inscription)
- Header sticky avec effet de scroll

#### Hero Section
- Titre attrayant avec gradient
- Sous-titre informatif
- Barre de recherche interactive
- Effets de parallaxe en arrière-plan

#### Cartes d'Anime
- Image du poster
- Overlay avec bouton de lecture
- Informations (titre, note, genre)
- Description courte
- Effet hover avec elevation

#### Sections
- **En Tendance**: Affichage 6 animes en grille
- **Les Plus Populaires**: Affichage 6 animes supplémentaires
- Animations d'apparition au scroll

#### Footer
- Sections: À propos, Support, Légal, Réseaux
- Liens de réseaux sociaux interactifs
- Copyright et informations

## 📱 Responsive Design

- **Desktop** (>1024px): Grille de 6 colonnes
- **Tablette** (768px-1024px): Grille de 4 colonnes
- **Mobile** (<768px): Grille de 2 colonnes
- **Petit Mobile** (<480px): Optimisation complète

## 🛠 Technologies

- **HTML5**: Structure sémantique
- **CSS3**: Gradients, flexbox, grid, animations
- **JavaScript**: Interactivité sans dépendances externes
- **Font Awesome 6.4**: Icônes vectorielles
